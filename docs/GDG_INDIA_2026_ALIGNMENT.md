# GDG India 2026 Alignment

## Problem Fit
RescueLink addresses emergency response coordination for:
- Disaster and crisis events (flood, fire, storm, evacuation)
- Daily vulnerability cases (elderly fall, urgent medicine need)
- Safety incidents (women or child danger)

## Inclusion and Accessibility
- Text scale and high-contrast support
- Screen-reader announcement support
- Haptic and flash SOS feedback options
- One-tap emergency fallback call/SMS

## Responsible Matching
- AI suggests category, severity, and recommended skill
- Routing maps requests to relevant responder categories
- Off-duty authorities can be configured to receive all incident alerts

## Reliability in Low Connectivity
- Cloud-first request creation
- Local pending queue and retry sync
- Simulated mesh/satellite communication fallback for disaster drills

## Next Technical Milestones
- Admin review dashboard for authority verification
- Device-level transport plugins for real mesh and satellite where supported
- Optional server push notifications for truly immediate background alerts
